### 2. **需要修改的配置**

要让OpenClaw支持跨设备通信，需要：

#### 配置A：绑定到所有网络接口
```json
// 在openclaw.json中添加
"gateway": {
  "bind": "0.0.0.0",  // 改为0.0.0.0允许所有IP访问
  "port": 18789
}
```

#### 配置B：指定局域网IP
```json
"gateway": {
  "bind": "192.168.1.100",  // 你的局域网IP
  "port": 18789
}
```

### 3. **安全考虑**
```
bind: "127.0.0.1"     → 最安全，仅本机访问
bind: "192.168.1.100" → 较安全，局域网内访问  
bind: "0.0.0.0"       → 风险较高，所有网络接口
```
