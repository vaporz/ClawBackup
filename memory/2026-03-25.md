## 今日重点：火山引擎 Seedream 5.0 Lite API 研究

### 问题解决过程

1. **API Key 配置**
   - Google AI Studio 的 nanobanana pro 需要付费，无法使用
   - 改用火山引擎 Seedream 5.0 Lite

2. **API 调用**
   - 使用火山引擎 ARK API: `https://ark.cn-beijing.volces.com/api/v3/images/generations`
   - 认证方式: `Authorization: Bearer <API_KEY>`

3. **关键参数**
   - ✅ 正确参数: `image`（图生图）
   - ❌ 错误参数: `reference_images`（这是另一个API的参数）
   - 尺寸需 ≥ 3686400 像素

4. **参考图问题**
   - r2.dev 域名在火山引擎服务器上无法解析
   - 改用七牛云国内可访问的URL

payload = {
