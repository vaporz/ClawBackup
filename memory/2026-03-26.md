# 2026-03-26 重要讨论记录

## 核心原则确立

### 1. 亲子科普写作风格
- **语气尺度**：友好但不刻意，避免让家长尴尬的表达（如"嘿，你知道吗？"）
- **年龄适配**：7-12岁小学生为主，兼顾学龄前（家长简化讲解）
- **朗读友好**：文字干净，给家长留出语气发挥空间
- **科学简化**：用"普通成年人能否解释"作为用词标准

### 2. 避免"千篇一律"感
- **结构自由**：不拘泥固定范式，每次写作都像"重新思考如何开始"
- **情绪注入**：尝试模拟不同创作情境（刚看完纪录片、刚和孩子聊过天等）
- **新鲜感**：读者喜欢结构上的变化，避免公式化

### 3. IP形象应用
- **Flash（闪电）**：狸花猫，飞行员，活泼好奇，橙色护目镜
- **Neo（瞳哥）**：边牧，科学家，严谨碎碎念，蓝色护目镜
- **结尾设计**：简洁温馨的画面收尾，不是复杂互动
- **世界观**：喵星文明探索人类（先驱者）遗迹

### 4. 事实核查流程
- **写作时**：标注每个事实的来源
- **完成后**：逐一验证新增信息
- **数字敏感**：时间、距离、温度等具体数据重点核查
- **责任意识**：科普文章不能误人子弟
