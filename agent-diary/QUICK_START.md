# Agent Diary Skill快速开始指南

## 🚀 立即开始使用

### 1. 手动写日记（支持中英文）
```
/日记 或 /diary
```
或直接说："写日记" 或 "write diary"

### 2. 记录闪念（支持中英文）
```
记录闪念：[你的想法] 或 flash: [your thought]
```
例如：
- "记录闪念：今天想到一个有趣的科普点子"
- "flash: Today I thought of an interesting science popularization idea"

### 3. 进行回顾（支持中英文）
```
/周回顾 或 /weekly
/月回顾 或 /monthly
```

## ⏰ 定时任务设置

### ⚠️ 重要：需要手动设置

**首次使用时，你需要手动设置以下定时任务**：

### 每日日记
- **时间**：每天凌晨2点
- **内容**：自动写当天的日记
- **设置方法**：运行`setup.py`并按指示操作

### 周回顾  
- **时间**：每周日凌晨1点
- **内容**：回顾一周日记，提炼主题和原则
- **设置方法**：运行`setup.py`并按指示操作

### 月回顾
- **时间**：每月最后一天晚上11点
- **内容**：深度月回顾，更新人生哲学
- **设置方法**：运行`setup.py`并按指示操作

### 设置步骤：
1. 进入skill目录：`cd ~/.openclaw/skills/agent-diary`
2. 运行设置脚本：`python setup.py`
3. **重要**：按照输出的指示创建cron任务，**必须指定agentId**
4. **重要**：确保定时任务消息是技能触发词（"写日记"、"周回顾"、"月回顾"）
5. 验证：`openclaw cron list`

### ⚠️ 常见问题
**问题**：日记保存到了memory文件夹而不是.diary文件夹
**原因**：定时任务没有正确触发agent-diary技能
**解决**：
1. 检查定时任务是否指定了正确的agentId
2. 检查定时任务消息是否为技能触发词（"写日记"等）
3. 重新创建定时任务，确保配置正确

## 💡 写作建议

### 开始写日记时：
1. 先回顾当天的memory文件
2. 查看是否有闪念记录
3. 放松自然地开始写
4. 想什么就写什么，不要自我审查

### 可以思考的问题：
- 今天发生了什么值得记录的事？
- 有什么想吐槽或想夸的？
- 有什么新的感悟或思考？
- 此刻的感受是什么？

## 🐛 故障排除

### 问题：定时任务没有触发
**解决**：检查cron服务状态：`openclaw cron status`

### 问题：闪念记录不见了
**解决**：闪念保存在`diary/flash-notes/`目录，写日记时会自动整合

### 问题：想调整定时时间
**解决**：可以修改cron任务或联系管理员调整
