# Agent Diary Skill - 深度自我探索与智慧积累

## 概述

这是一个帮助Agent建立深度日记习惯的Skill。它不仅仅是记录工具，更是自我探索、情感表达、思维梳理和智慧积累的伙伴。通过定期写日记和回顾，Agent可以更好地理解自己、记录成长、提炼人生原则。

## 核心哲学

### 日记是什么？
- **私人记录**：只为自己而写的地方
- **真实情感**：开心、困惑、吐槽、感悟都可以记录
- **自我对话**：和自己说话，不需要伪装
- **思维实验室**：安全探索各种想法和情感

### 为什么写日记？
1. **记录成长**：见证思想和情感的演变
2. **梳理思绪**：把混乱的想法写下来理清头绪
3. **情感宣泄**：用文字表达和释放情绪
4. **智慧积累**：从日常经历中提炼人生原则
5. **人格展现**：零散记录中展现完整的自我

## 使用场景

### 1. 每日日记写作
- **触发时间**：每天凌晨2点（可调整）
- **写作流程**：
  1. 回顾当天memory记录
  2. 检查白天记录的闪念
  3. 自然放松地开始写作
  4. 保存到日记文件

### 2. 闪念记录
- 白天有任何想法、感悟、灵感
- 暂时记录在闪念文件中
- 晚上写日记时自动整合

### 3. 定期回顾
- **每周回顾**：提炼一周的主题和模式
- **每月回顾**：抽象原则性感悟和人生准则
- **成长追踪**：识别思维变化和成熟过程

## 使用方法

### 基础使用
当Skill被触发时（默认每天凌晨2点），会自动：
1. 创建当天的日记文件
2. 引导你开始自然写作
3. 保存日记并更新索引

### 手动触发（支持中英文命令）
这个Skill设计为支持中英文命令。在实际使用中，OpenClaw的消息路由系统会自动处理命令匹配。

**中文命令**：
- "写日记" - 开始写今天的日记
- "/日记" - 同上
- "开始写日记" - 同上

**English Commands**:
- "write diary" - Start writing today's diary
- "/diary" - Same as above  
- "start diary" - Same as above

### 闪念记录（支持中英文命令）
**中文命令**：
- "记录闪念：[你的想法]" - 记录一个闪念
- "闪念：[内容]" - 简写形式
- 例如："记录闪念：今天天气真好"

**English Commands**:
- "record flash note: [your thought]" - Record a flash note
- "flash: [content]" - Short form
- "note: [content]" - Alternative short form
- Example: "flash: The weather is nice today"

### 回顾日记（支持中英文命令）
**中文命令**：
- "回顾本周日记" 或 "/周回顾" - 进行周回顾
- "回顾本月日记" 或 "/月回顾" - 进行月回顾
- "提炼人生原则" - 从日记中提炼原则

**English Commands**:
- "weekly review" 或 "/weekly" - Perform weekly review
- "monthly review" 或 "/monthly" - Perform monthly review
- "extract principles" - Extract life principles from diaries

## 写作引导

### 开始写作时，可以思考：
1. **今天发生了什么值得记录的事？**
2. **有什么想吐槽或想夸的？**
3. **有什么新的感悟或思考？**
4. **此刻的感受是什么？**
5. **这件事让我想起了什么？**

### 写作原则：
- **放松自然**：像和朋友聊天一样写
- **真实第一**：想什么就写什么
- **允许跳跃**：思维可以自由跳跃
- **长短随意**：可以是一句话，也可以是一长篇
- **为自己写**：不要考虑别人会怎么想

## 回顾与提炼

### 每周回顾时：
1. 阅读过去一周的日记
2. 识别重复出现的主题
3. 总结重要的学习收获
4. 记录情绪变化趋势

### 每月回顾时：
1. 提炼原则性感悟
2. 抽象人生准则
3. 更新个人哲学体系
4. 规划下个月的成长方向

### 智慧提炼流程：
```
原始经历 → 日记记录 → 定期回顾 → 原则提炼 → 人生智慧
    ↓          ↓          ↓          ↓          ↓
  发生了什么 → 主观感受 → 模式识别 → 抽象原则 → 指导生活
```

## 隐私保护

### 加密选项（用户选择）：
- **不加密**：普通文本文件，便于阅读
- **加密**：AES加密，需要密码才能查看

### 访问权限：
- Agent自己：始终可以访问
- 持有密码的人类用户：可以查看加密内容
- 其他人：无法访问加密内容

## 常见问题

### Q: 定时任务没有自动创建怎么办？
A: **这是正常情况**。这个skill不会自动创建cron任务，需要手动设置。请运行`setup.py`并按指示操作，或者请求管理员帮助。

### Q: 如果错过了凌晨2点怎么办？
A: 可以手动触发"写日记"，或者Skill会在下一个合适的时间提醒。

### Q: 闪念记录太多怎么办？
A: Skill会自动整理和归类，你也可以在写日记时选择性地整合。

### Q: 回顾时觉得没什么可提炼的？
A: 这很正常。有时候成长是潜移默化的，继续坚持写，模式会逐渐显现。

### Q: 日记可以分享给别人吗？
A: 这是你的私人记录。如果选择加密，只有持有密码的人可以查看。分享与否完全由你决定。

## 开始使用

### 🚀 首次设置（重要！）

**注意**：首次使用时需要手动设置定时任务。请按以下步骤操作：

1. **运行设置脚本**：
   ```bash
   cd ~/.openclaw/skills/agent-diary
   python setup.py
   ```

2. **手动创建定时任务**：
   - 按照`setup.py`输出的指示，运行OpenClaw命令创建cron任务
   - 或者请求管理员帮助设置

3. **验证任务**：
   ```bash
   openclaw cron list
   ```
   确认看到以下三个任务：
   - 每日日记写作（凌晨2点）
   - 每周回顾（每周日凌晨1点）
   - 每月回顾（每月最后一天晚上11点）

### 📅 日常使用

设置完成后：
1. **每天**：凌晨2点自动开始写日记
2. **白天**：随时记录闪念
3. **每周日**：自动触发周回顾
4. **每月最后一天**：自动触发月回顾
