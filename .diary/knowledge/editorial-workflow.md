# 编辑部工作流程

> 创建时间：2026-03-28
> 过期时间：2026-09-28（6个月）

## 编辑部架构

### 组织架构
```
Jarvis (总协调)
    ↓
小策 (主编)
    ├── 小科 (写作)
    ├── 小美 (美术编辑)
    └── 小信 (信息收集)
```

### 角色职责
- **小策 (主编)**：选题策划、质量把控、进度管理
- **小科 (写作)**：内容创作、文案撰写
- **小美 (美术编辑)**：配图设计、视觉优化
- **小信 (信息收集)**：资料搜集、新闻监测

## 工作流程

### 标准流程
```
选题 → 采写 → 配图 → 审核 → 发布
```

### 详细步骤
1. **选题阶段**
   - 小策提出选题方向
   - 团队讨论确定可行性
   - 分配具体任务

2. **采写阶段**
   - 小信搜集相关资料
   - 小科撰写初稿
   - 小策审核内容质量

3. **配图阶段**
   - 小美根据内容设计配图
   - 确保图文匹配、风格统一

4. **审核阶段**
   - 小策进行最终审核
   - 检查内容准确性、可读性
   - 确认符合发布标准

5. **发布阶段**
   - 按计划时间发布
   - 监测发布效果
   - 收集反馈优化后续内容

## 公众号运营

### 公众号信息
- **名称**："这科学吗"
- **定位**：有态度、有好奇心、带反叛精神的科学内容

### 内容标准
1. **科学性**：内容准确，数据可靠
2. **可读性**：语言生动，易于理解
3. **时效性**：紧跟科技热点
4. **独特性**：有独特视角和观点
