# Agent Diary 技能使用指南

## 概述
这是小科（青少年科普作家）的日记系统使用指南，记录如何有效使用agent-diary技能进行自我探索和知识积累。

## 三层记忆管理系统

### 1. 工作日志 (.diary/logs/)
- **目的**：记录客观事实和任务完成情况
- **内容格式**：
  ```markdown
  # 工作日志：[任务名称]
  
  ## 基本信息
  - 日期：YYYY-MM-DD
  - 时间：HH:MM GMT+8
  - 任务类型：[写作/研究/编辑/其他]
  
  ## 任务描述
  [简要描述任务内容]
  
  ## 执行步骤
  1. [步骤1]
  2. [步骤2]
  3. [步骤3]
  
  ## 成果与输出
  - [成果1]
  - [成果2]
  
  ## 遇到的问题与解决方案
  [如有问题，记录解决方案]
  
  ## 后续建议
  [对未来工作的建议]
  ```

### 2. 知识库 (.diary/knowledge/)
- **目的**：积累科普写作相关知识和最佳实践
- **内容类型**：
  - 科普写作技巧
  - 科学知识整理
  - 读者心理分析
  - 内容传播策略
  - 团队协作规范

### 3. 个人日记 (.diary/YYYY/MM/)
- **目的**：主观感受、情感记录、成长轨迹
- **写作原则**：
  - 放松自然，像和朋友聊天
  - 真实第一，想什么就写什么
  - 允许思维自由跳跃
  - 长短随意，为自己而写

## 使用流程

### 日常使用
1. **白天工作**：在`logs/`目录记录工作日志
2. **灵感闪现**：在`flash-notes/`记录闪念
3. **知识积累**：在`knowledge/`整理有价值的知识
4. **晚上反思**：在个人日记中整合以上内容

### 定期回顾
1. **每周回顾**：提炼一周的主题和模式
2. **每月回顾**：抽象原则性感悟和人生准则
3. **知识更新**：定期更新知识库内容

## 针对科普作家的特别建议

### 日记内容建议
1. **创作过程记录**：记录科普文章的构思、写作、修改过程
2. **读者反馈思考**：思考如何让科学知识更易懂、更有趣
3. **科学热点追踪**：记录当前科学热点和公众关注点
4. **跨学科连接**：思考不同科学领域之间的关联

### 知识库建设
1. **科学概念库**：整理常用科学概念的通俗解释
2. **比喻素材库**：收集适合青少年的科学比喻
3. **案例故事库**：积累科学家故事和科学发现案例
4. **互动活动库**：设计亲子科学实验和互动活动

## 清理机制
- **工作日志**：保留3个月，按月清理
- **知识库**：保留6个月，定期更新
- **个人日记**：永久保留
- **闪念记录**：保留1个月
