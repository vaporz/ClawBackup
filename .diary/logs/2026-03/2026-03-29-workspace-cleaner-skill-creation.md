## 技能特性

### 1. 智能文件识别
- **核心文件**：身份文件、团队文件（永久保留）
- **临时文件**：.tmp, .log, .bak等（归档保留）
- **版本文件**：同一文件多个版本（保留最新版）
- **草稿文件**：未完成的工作（归档保留）

### 2. 安全清理机制
- **回收站系统**：删除文件先到回收站，保留30天
- **元数据记录**：记录删除时间、原路径等信息
- **恢复支持**：支持误删文件恢复
- **自动清理**：回收站过期文件自动清理

### 3. 归档管理系统
```
归档/
├── YYYY-MM/                    # 按月组织
│   ├── drafts/                # 草稿文件
│   ├── tests/                 # 测试文件
│   ├── versions/              # 版本历史
│   └── temp/                  # 临时文件
└── recycle-bin/              # 回收站
```

### 4. 多种清理策略
- **strict（严格）**：只保留核心文件
- **balanced（平衡）**：保留核心文件和近期工作文件（默认）
- **loose（宽松）**：保留较多文件

### 5. 定时任务支持
- 可配置清理频率（每天/每周/每月）
- 默认：每周日晚上10点自动清理
- 后台静默执行，不影响工作

2. **识别文件类型**：基于文件名、内容、时间戳
3. **应用清理规则**：根据配置决定处理方式
4. **执行清理操作**：保留、归档、删除
5. **生成报告**：统计清理结果，识别问题
