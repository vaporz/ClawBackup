## 🔧 技术细节

### 微调5的精确分类逻辑
```python
# 精确判断段落类型
is_title = ("一、" in text or text.startswith("1."))
is_list = text.startswith("•") or text.startswith("- ")
is_image_caption = "图" in text and "：" in text

# 只对普通正文段落设置缩进
if not (is_title or is_list or is_image_caption):
    para.paragraph_format.first_line_indent = Cm(1.1)
else:
    # 标题和列表保持0缩进
    para.paragraph_format.first_line_indent = Cm(0)
```

### 格式检查1的标题修复
```python
# 公文标题：小标宋，22pt，居中
if text == "勘星科技航天模拟体验项目运营说明":
    for run in para.runs:
        run.font.name = "小标宋"
        run.font.size = Pt(22)
    para.paragraph_format.alignment = WD_ALIGN_PARAGRAPH.CENTER

# 一级标题：黑体，16pt
elif "一、" in text:
    for run in para.runs:
        run.font.name = "黑体"
        run.font.size = Pt(16)

# 二级标题：楷体_GB2312，16pt，加粗
elif text.startswith("1."):
    for run in para.runs:
        run.font.name = "楷体_GB2312"
        run.font.size = Pt(16)
        run.font.bold = True
```
