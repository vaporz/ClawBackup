# Workspace Cleaner - 快速开始指南

## 🚀 5分钟上手

### 1. 安装skill
```bash

clawhub install workspace-cleaner

# 或手动安装
cd ~/.openclaw/skills
git clone [仓库地址] workspace-cleaner
```

### 2. 快速配置
```bash
cd ~/.openclaw/skills/workspace-cleaner
python setup.py --quick
```

```bash

```

## ⚙️ 配置选项

### 清理策略
```yaml
# config.yaml
strategy: "balanced"  # strict（严格）, balanced（平衡）, loose（宽松）
```

### 文件保留规则
```yaml
retention:
  core_files: "keep"           # 身份文件、团队文件等
  temp_files: "archive_30d"    # 临时文件归档保留30天
  version_files: "keep_latest" # 只保留最新版本
```

### 定时任务
```yaml
scheduled:
  enabled: true
  frequency: "weekly"          # daily, weekly, monthly
  day_of_week: "sunday"        # 周日执行
  time: "22:00"               # 晚上10点
```

## 🆘 常见问题

### Q: 如何恢复误删的文件？
A: 使用`管理回收站`命令，找到文件后选择恢复。

### Q: 清理会影响正在使用的文件吗？
A: 不会。skill会检查文件是否被占用，避免影响工作。

### Q: 可以清理其他目录吗？
A: 可以，使用`清理工作区 --path=/自定义/路径`。

### Q: 如何调整清理强度？
A: 使用`--strategy`参数：strict（严格）、balanced（平衡）、loose（宽松）。
