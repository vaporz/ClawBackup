# MEMORY.md - 编辑部长期记忆

## 公众号信息

- **名称**：这科学吗
- **定位**：青少年科普公众号
- **目标读者**：7-12岁小学生 + 家长（亲子共读）
- **核心价值**：让科学变得有趣易懂，促进亲子互动学习

### 内容风格
- 语气友好但不刻意，避免让家长尴尬的表达
- 年龄适配7-12岁，兼顾学龄前
- 文字干净朗读友好，给家长留出语气发挥空间
- 用"普通成年人能否解释"作为用词标准
- 结构自由不拘泥固定范式，避免公式化
- 情绪注入，模拟不同创作情境

### 选题思路（啸指导）
- 小信的资料是创意起点，不是天花板
- 从热点新闻发散联想，找关联科普话题
- 用"然后呢"思维：读者看到热点后会好奇什么？往深里挖
- 找共鸣点：把专业话题跟普通人生活联系起来，降低距离感
- 热点是钩子，深度科普是内容，亲子共读是场景
- 例："航天员在空间站烤鸡翅" → 航天员食谱 / 太空食品发展 / 太空技术对普通人饮食的影响

### 选题方法论（啸指导 + 实践验证）
**"一条新闻 → 一个话题宇宙"**
1. 小信给种子素材
2. 我做发散联想，每个种子展开3-5个方向
3. 筛出适合"这科学吗"定位的选题
4. 分配给小科写、小美配图

**核心能力：发散思维**
- 科普的本质不是知道答案，而是会问问题
- 从一个点切入，可以展开很多有趣的话题
- 这是科普专家的核心能力，也是选题策划的关键
